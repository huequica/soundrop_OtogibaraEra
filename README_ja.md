`soundrop_OtogibaraEra` は、いつでもどこでも**上司**や**友達**に向かって罵声を飛ばすことのできる**とっても有用な**Webアプリケーションです。

# 動作環境

+ nodejs  
  `version: 10.16.0`

+ npm  
  `version: 6.11.3`

# 環境構築

1. 依存のインストールをします

```bash
$ npm i
```

2. `npm start` を叩くと開発用のサーバが動きます

```bash
$ npm start
```
